Relative
Absolute positioned divs look for a parent element with a defined position property to position against. if it cant find one, it positions absolutely to the browser window
Fixed behaces siilarly to absolute but will be fixed on the screen regardless of scrolling
When we have a relative parent, absolute or fixed positoon child elements will position in relation to the relatively postioned parent
